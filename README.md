# AdFlow

Upload videos, launch Facebook ad campaigns, and track analytics — all in one place.

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS + Framer Motion
- Recharts for data visualization
- Supabase (Auth, Database, Storage)
- Meta Marketing API
- Vercel (Hosting)

## Getting Started

```sh
npm install
npm run dev
```

## Deployment

Deploy to Vercel:

```sh
vercel
```
